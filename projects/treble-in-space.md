In my MUS400 "Adaptive Music Composition" course, we learned how to write music, and how to transfer it to FMOD. Using FMOD, I learned how to change the music based off of given boolean or integer inputs. We worked with the ICS 485 and ACM 487 class "Video Game Design & Development" to create video games with adaptive music.

Unfortunately, this was the semester when the COVID-19 pandemic first arrived, so that made collaboration difficult, but we still found a way to work on it and get a final product. This class was a very interesting look into how music can blend with ICS and video game development in a very immersive way.

The final product can be found [here](https://www.youtube.com/watch?v=yFQkSQ-_p7w).
